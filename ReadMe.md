# What is project about
Project is created for storing file's hashes in blockchain format to verify integrity of files
# Usage
## For GUI
```console
./main gui
```
## For TUI
```console
./main tui
```
# How to build it
## Libraries required
This project require OpenSSL to work
```console
sudo apt install openssl 
```
## Using CMake
```console
mkdir build 
cd build 
cmake .. 
make 
```
